  • Классы 
    Класс Описание
    _AdvancedRadiansRobot
    Используется системой в качестве замены для всех вызовов *Radians в AdvancedRobot.
    _AdvancedRobot
    Используется системой, а также является заменой для всех устаревших вызовов для AdvancedRobot.
    _Robot
    Используется системой, а также является заменой для всех устаревших вызовов для Robot.
    _RobotBase
    Базовый класс всех роботов, используемых системой.
    AdvancedRobot
    Более продвинутый тип робота, чем Robot, который позволяет выполнять неблокирующие вызовы, создавать пользовательские события и записывать данные в файловую систему.
    BattleEndedEvent
    Отправляется в onBattleEnded() когда битва закончена.
    BattleResults
    Содержит результаты битвы, возвращаемые BattleEndedEvent.getResults() когда битва закончена.
    BattleRules
    Содержит правила битвы, возвращаемые BattleStartedEvent.getBattleRules() когда битва начинается и BattleCompletedEvent.getBattleRules() когда битва завершена.
    Bullet
    Представляет пулю.
    BulletHitBulletEvent
    Это событие отправляется в onBulletHitBullet когда пуля одного из ваших роботов попадает в другую пулю.
    BulletHitEvent Это событие отправляется в onBulletHit() когда пуля одного из ваших роботов попадает в другого робота.
    BulletMissedEvent
    Это событие отправляется в onBulletMissed() когда пуля одного из ваших роботов промахивается, т.е. когда пуля достигает границы поля боя.
    Condition
    Используется для определения пользовательских условий с помощью waitFor(Condition) и пользовательских событий для AdvancedRobot.
    CustomEvent
    Это событие отправляется в onCustomEvent() когда пользовательское условие выполнено.
    DeathEvent
    Это событие отправляется в onDeath() когда ваш робот умирает.
    Event
    Суперкласс всех событий Robocode.
    GunTurnCompleteCondition
    Предварительно построенное условие, которое вы можете использовать, указывающее, что орудие закончило поворот.
    HitByBulletEvent
    Это событие отправляется в onHitByBullet() когда ваш робот поражен пулей.
    onHitRobot()
    Это событие отправляется в когда ваш робот сталкивается с другим роботом.
    HitWallEvent
    Это событие отправляется в onHitWall() когда вы сталкиваетесь со стеной.
    JuniorRobot
    Это самый простой тип робота, который проще, чем Robot и AdvancedRobot классов.
